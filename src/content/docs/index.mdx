## Features

<CardGrid stagger>
	<Card title="Insane Performance" icon="rocket">
		Tiny VPS down to 128MB RAM and 1 vCPU is enough to handle all requests if with 1Gbps bandwidth limit.
	</Card>
	<Card title="Easy to Deploy" icon="setting">
		All you need is just `cargo install rlist-cli`. Docker image is also available.
	</Card>
	<Card title="External Feather" icon="external">
		Configure captcha, analytics, and more with external feathers in default `config.json` without any code.
	</Card>
	<Card title="Atlas" icon="open-book">
		Users can use [rList Atlas]() to subscribe all sharers and find everything that they need.
	</Card>
